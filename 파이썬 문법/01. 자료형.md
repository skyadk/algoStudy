## 리스트 자료형

### 리스트 만들기

```python
list1 = list()
list2 = []
a = [ i for i in range(1,11)] = list(range(1,11))
print(a[-2]) #뒤에서두번째원소
print(a[2:5]) #2,3,4번째원소출력
b = [ i for i in range(1,10) if i%2 == 1]
a.append(11) #맨뒤에 11추가
a.reverse() = a[::-1]  #뒤집기 
a.insert(0,12) #인덱스0자리에 12원소 추가
a.count(2) # a에 2의 갯수 카운트
list[::2] = [2,4,6,8,10]


```
